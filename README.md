# 🎓 Complaint Management System (ASP.NET)

## 📌 Project Description
This project is a Complaint Management System developed using ASP.NET Web Forms and C#.  
It helps students to submit complaints and allows Admin and HOD to manage and resolve them efficiently.

The system provides role-based access for Students, Admin, and HOD to ensure proper complaint handling and tracking.

---

## 🚀 Features
- 👤 User Registration & Login
- 📝 Submit Complaints
- 📊 Student Dashboard
- 🛠️ Admin Panel to manage complaints
- 🎯 HOD Dashboard for monitoring
- 🔄 Complaint Status Tracking
- 🗂️ Database Integration

---

## 🛠️ Technologies Used
- ASP.NET Web Forms
- C#
- SQL Server
- ADO.NET
- HTML, CSS

---

## 📂 Project Structure
- **Student Module** → Submit and track complaints  
- **Admin Module** → Manage and update complaints  
- **HOD Module** → Monitor complaint status  

---

## ⚙️ How to Run the Project

1. Install Visual Studio  
2. Open the project folder  
3. Open `.sln` file in Visual Studio  
4. Configure SQL Server Database  
5. Run the project using IIS Express
